Welcome to Hvuntokrul's Image Search App!

Use https://fcc-ims.herokuapp.com/api/YOUR_SEARCH_REQUEST link format with the API:
e.g. https://fcc-ims.herokuapp.com/api/zen

You can supply an offset to view other (less relevant) search results, e.g.
https://fcc-ims.herokuapp.com/api/zen?offset=5

You can also see the latest search results by going to:
https://fcc-ims.herokuapp.com/latest/
